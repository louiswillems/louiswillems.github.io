Le Guide du Data Pro

Bienvenue dans ce guide des outils pour la data 👋

Ce guide a été conçu pour vous guider à travers les outils et les bonnes pratiques essentiels pour travailler efficacement dans le domaine de la data. Que vous soyez débutant ou que vous souhaitiez approfondir vos connaissances, ce guide vous accompagnera pas à pas.

Objectifs du guide#

  • Comprendre et configurer les outils de développement comme VSCode et Git.

  • Apprendre à utiliser le terminal pour automatiser vos tâches.

  • Découvrir les bases de Python et la gestion des environnements virtuels.

  • Explorer les concepts avancés comme le versionnement de code et les workflows collaboratifs.

À qui s’adresse ce guide ?#

Ce guide s’adresse à :

  • Les étudiants en data science ou en informatique.

  • Les professionnels souhaitant améliorer leur productivité.

  • Toute personne curieuse d’apprendre les bases des outils modernes de développement.

Structure du guide#

Le guide est organisé en chapitres, chacun couvrant un sujet spécifique. Cliquez sur les liens ci-dessous pour accéder directement aux chapitres :

  • 1. Installation de VSCode

  • 2. Personnalisation du terminal

  • 3. Utilisation du terminal

  • 4. Installation de Python

  • 5. Gestion de version et packaging Python

  • 6. Environnements virtuels

  • 7. Configuration de VSCode

  • 8. Installation de Git

  • 9. Utilisation de Git

  • 10. Git local vers Git collaboratif

  • 11. Utilisation avancée de Git

Nous espéros que ce guide vous sera utile et qu’il vous aidera à devenir un expert dans l’utilisation des outils du data pro.

Bonne lecture !
